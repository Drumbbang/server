### 💡 Git Branch 정책 `git flow 활용`
- Main : 배포시 사용
- Develop : 개발이 끝난 부분에 대해서 Merge 진행
- Feature : 기능 개발을 진행할 때 사용
- Release : 배포를 준비할 때 사용
- HotFix : 배포를 진행한 후 발생한 버그를 수정해야 할 때 사용
